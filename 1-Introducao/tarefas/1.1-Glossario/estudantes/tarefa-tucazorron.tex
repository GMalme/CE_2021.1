\newglossaryentry{DevOps}{
    name={DevOps},
    description={DevOps é um conjunto de práticas que combinam desenvolvimento de software com operações de TI. Tem como objetivo encurtar o ciclo de vida de desenvolvimento de sistemas e fornecer entrega contínua com alta qualidade de software. O DevOps é complementar ao desenvolvimento de software Agile. Vários aspectos do DevOps vieram da metodologia Agile. Fonte: \cite{wikipedia_devops_2020}}}
