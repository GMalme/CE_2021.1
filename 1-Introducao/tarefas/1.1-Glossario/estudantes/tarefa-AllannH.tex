\newglossaryentry{MetodoExperimental}{
name = {Método experimental},
description  = {
Processo de observação baseado em hipótese, experimentação, descrição e resultado para garantir a integridade do método. Consiste na alteração de variáveis independentes e analise do resultado obtido após a mudança comparado com variáveis que tenham relações diretas com o objeto examinado anteriormente.
Caso concreto: pesquisas cientificas para conhecimento do universo.
Fonte: \cite{rabelo_o_2011}.
}
}
