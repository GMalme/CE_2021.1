\newglossaryentry{AnaliseLogica}{
	name={AnaliseLogica},
description={A análise lógica não é uma análise de elementos existentes. É o traçado de relações entre conceitos na suposição de que junto com cada conceito dado ou encontrado é dado seu negativo, e todas as outras relações resultantes de uma transposição de seus correlatos. Este último postulado consiste apenas em identificar cada correlato e distingui-lo dos demais sem reconhecer nenhuma ordem serial entre eles.
Fonte: \cite{commens_logical_2021}
}
}
