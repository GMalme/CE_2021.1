name = {Cientometria (\textit{Scientometrics})},
description  = {
Cientometria é definida como o estudo da mensuração e quantificação do progresso científico, estando a pesquisa baseada em indicadores bibliométricos. Tem sido usada para ajudar nações a tomarem decisões sobre quais áreas do saber necessitam de fundos especiais.
