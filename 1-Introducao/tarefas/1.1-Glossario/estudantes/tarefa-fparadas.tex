\newglossaryentry{Hipotese}{
name = {Hipótese},
description  = {
Uma hipótese é uma explicação proposta para um fenômeno. Para que uma hipótese seja científica, o método cientifico requer que seja capaz de ser testada. Cientistas geralmente baseiam hipóteses científicas em observações prévias as quais não possuem explicações satisfatórias nas teorias científicas disponíveis.
Fonte: \citet{wikipedia_hypothesis_nodate}.
}
}
