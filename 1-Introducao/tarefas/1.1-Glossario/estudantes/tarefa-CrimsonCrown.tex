\newglossaryentry{ReprodutibilidadeCientifica}{
name = {ReprodutibilidadeCientifica},
description  = {
É o princípio que diz que qualquer experimento que gera conhecimento científico deve ser reproduzível por outros pesquisadores que utilizarem o mesmo método e dados. Esse princípio ajuda a evitar o efeito de acidentes durante a experimentação e bias dos cientistas. Fonte: \cite{noauthor_reproducibility_nodate}.
}
}
